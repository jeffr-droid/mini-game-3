# Kleines Spiel

Ein einfaches Springspiel, bei dem du Hindernisse vermeiden musst.

## Wie man spielt
- Öffne `index.html` in einem Webbrowser.
- Drücke die Leertaste (auf Desktop) oder tippe auf den Bildschirm (auf Mobilgeräten), um zu springen.
- Vermeide die roten Hindernisse.
- Nach 3 erfolgreichen Durchläufen erscheint ein goldener Pokal, den du erreichen musst, um zu gewinnen und deine E-Mail einzugeben.

## Anforderungen
- Ein moderner Webbrowser (Chrome, Firefox, Edge, etc.) oder mobiles Gerät mit Touchscreen.

## Troubleshooting
- Stelle sicher, dass JavaScript aktiviert ist.
- Wenn das Spiel nicht lädt, überprüfe die Dateiberechtigungen.
